The Problem

Cloud trivia apps like Kahoot collapse the moment 200 people hit a venue's congested Wi-Fi. You get spinning loaders, disconnected players, and latency spikes that make live quiz nights feel like a buffering video. The server is somewhere in Ohio. Your audience is right here.

The Solution

LocalFlux bypasses the cloud entirely. It runs natively on a local 192.168.x.x router, so every packet travels a few metres — not thousands of kilometres. The result is guaranteed 0 ms latency for physical crowds, zero dependency on the internet, and a server you can hold in your hand.

Core Features

🔒

Dumb Client Anti-Cheat

Answers never leave the server until the timer ends. The client is intentionally kept dumb — there's nothing to inspect, intercept, or brute-force.

🚪

The VIP Bouncer

A connection queue system that protects the physical router from hardware crashes. Players wait in line — the router never gets flooded.

🃏

Deck Studio

A browser-based CSV-to-JSON trivia forge. Drop in a spreadsheet, get a ready-to-play question deck. No CLI knowledge required.

Quick Start

bash 
# 1. Clone the repo
git clone https://github.com/Unknownbeliek/foss-hack-quiz-engine.git
cd foss-hack-quiz-engine

# 2. Start the server
cd server && npm install && npm start

# 3. Start the client (new terminal)
cd client && npm install && npm run dev

Then open http://localhost:5173 on every device on the same LAN.
